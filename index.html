Planned direction

Template options first, builder controls later.

The next version can evolve from static HTML options into a lightweight builder where the user can enter a recipient name and choose the right template type before copying or downloading the final email.

Name field

Future templates can include a field for a recipient name, then use that name in the opening line of the email.

Template types

Future options can include trust, individual, business, and other investor-specific versions.

Copy or export

Each generated email should be easy to preview, copy as a rendered email body, or copy as source HTML.
